Java Fullstack Practice
Practice repository from my journey learning Java and full-stack web development. This is a personal learning log — files here are exercises, not production projects.
What's inside
Java fundamentals
Variables, operators, data types (Var.java, Operator.java, Bitwise.java, assignopr.java)
Conditionals & loops (Conditional.java, Loops.java, Comparision.java)
Number pattern problems (NumberPatt.java, Pattern1.java, Patt.java)
Classic algorithm practice (Armstrong.java, Palindrome.java, Disarium.java, Strong.java, PrimPal.java, SumDig.java)
Recursion (Recursion.java, Recur.java, Recur1.java, Recur2.java)
OOP basics — static, instance, methods (Static.java, Instance.java, Method.java)
Exception handling (Multicatch.java)
Misc practice programs (Salary.java, Emp.java, Quadratic.java, Largest.java, Leap.java, Swap.java, and others)
Web basics (HTML/JS)
Simple interactive pages: calculator.html, loginform.html, regform.html, resume.html
JS concept practice: basicjs.html, assignjs.html, arthoper.html, evenjs.html
Goal
Building a strong foundation in Java (syntax, OOP, problem-solving) alongside basic front-end web skills, as a stepping stone toward full-stack development (Java backend + web frontend).
