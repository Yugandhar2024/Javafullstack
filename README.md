## 🎯 Learning Paths

### **Path 1: Java Fundamentals → OOP** (Beginner)
1. **Start Here**: `src/basics/`
   - Learn variables, data types, operators
   - Understand control flow (if-else, switch)
   - Master loops (for, while, do-while)

2. **Next**: `src/oop/`
   - Static vs Instance variables
   - Method definitions and calls
   - Class instantiation and object references

3. **Practice**: `src/algorithms/`
   - Apply fundamentals to solve real problems
   - Strengthen logical thinking

## 🚀 Getting Started

### Prerequisites
- **Java 8+** installed
- **Text Editor** or **IDE** (VS Code, IntelliJ, Eclipse)
- **Web Browser** (for HTML/JS files)

### Run Java Programs
```bash
# Navigate to src directory
cd src/basics

# Compile
javac VariableDemo.java

# Run
java VariableDemo
```

### View Web Projects
```bash
# Open in browser (or use Live Server extension in VS Code)
open web/interactive/Calculator.html
```

## 💡 Pro Tips

1. **Run with Input**: Many programs use `Scanner`. Run them in terminal for interactive input:
   ```bash
   java ArmstrongNumber
   # Then enter a number when prompted
   ```

2. **Experiment**: Modify numbers, patterns, conditions, and observe outputs.

3. **Trace Execution**: Use print statements or a debugger to trace loops and recursion.

4. **Compare Approaches**: Look at similar files to see different solutions to the same problem.

5. **JavaScript Testing**: Open web files in browser console (F12) to see debug output.
